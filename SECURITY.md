# Security Policy

## Reporting a Vulnerability

Please report security issues privately by email:

- security@ioi.dev

Include the following details when possible:

- Affected package and version
- Reproduction steps or proof of concept
- Potential impact and suggested remediation

We will acknowledge reports as quickly as possible and coordinate a fix before
disclosure.

## Supported Versions

At this stage, only the latest `main` branch is supported for security fixes.
